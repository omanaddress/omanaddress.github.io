v-bind给src和href赋值

用v-bind给src和href赋值其实很简单，即 v-bind:属性名="name"，其中name就是data里json数据的键值，其简写形式为 :属性名="name"。

示例

 
          link          
          link1          
          link2          
          啦啦啦          
                
            
var ve2 = new Vue({      el: '#app',      data: {          link: "linkyou.html",          link1: "https//www.baidu.com",          link2: "www.baidu.com",          img: '8.jpg'      }  })
