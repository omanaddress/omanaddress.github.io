MySQL GET_FORMAT()函数的用途是什么？

  • 首页
  • 文章分类
  • 正文

MySQL GET_FORMAT()函数用于根据作为第二个参数给出的标准格式，以格式化的方式转换DATE，TIME，DATETIME或TIMESTAMP之类的数据类型。标准格式类型可以是EUR，INTERNAL，ISO，JIS或USA。返回的格式代码与DATE_FORMAT()函数使用的代码相同。此函数与DATE_FORMAT()和STR_TO_DATE()函数结合使用非常有用。

语法

GET_FORMAT(data_type, standard_format)

就像我们前面讲的，data_type可以是DATE，TIME，DATETIME或TIMESTAMP，而standard_format可以是EUR，INTERNAL，ISO，JIS或USA。

示例

mysql> Select GET_FORMAT(DATE, 'USA') AS 'AMERICAN Format', -> GET_FORMAT(DATE, 'ISO') AS 'ISO Format';  +-----------------+------------+ | AMERICAN Format | ISO Format | +-----------------+------------+ | %m.%d.%Y | %Y-%m-%d          | +-----------------+------------+ 1 row in set (0.00 sec)

上面的查询返回美国使用的日期格式和ISO日期格式。

mysql> Select GET_FORMAT(TIME, 'USA') AS 'AMERICAN Format', -> GET_FORMAT(TIME, 'ISO') AS 'ISO Format';  +-----------------+------------+ | AMERICAN Format | ISO Format | +-----------------+------------+ | %h:%i:%s %p | %H:%i:%s       | +-----------------+------------+ 1 row in set (0.00 sec)

上面的查询返回美国使用的时间格式和ISO日期格式。
