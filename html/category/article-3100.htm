docker system

docker system --help 查看帮助

docker system df
提供Docker整体磁盘使用率的概况，包括镜像、容器和（本地）volume。所以我们现在随时都可以查看Docker使用了多少资源。

TYPE 类型
TOTAL 总数
ACTIVE 正在运行
SIZE 暂用磁盘大小
RECLAIMABLE 可回收的（未被使用的）

查看容器使用卷的情况和大小

docker system df -v

docker system info（就等同于 docker info）

docker system events（就等同于docker events）
获取docker系统实时事件，不包括容器内的

截止（日期时间）之前的操作

docker system events --until 时间

从（日期时间）之后的操作

docker system events --since 时间

docker system prune
修剪一切
docker system prune 命令是修剪镜像、容器和网络的快捷方式，在Docker 17.06.1之后要清理卷的话需要加上 --volumes

docker system prune --volumes		#使用-f 强制删除
